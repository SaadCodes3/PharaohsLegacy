# 🏺 PharaohsLegacy

A modern interactive website about Ancient Egyptian Civilization built with React and Vite.

Explore the greatness of ancient Egypt through:
- Famous Pharaohs
- Legendary Temples
- Egyptian Museums
- Historical Timeline
- Tourism Booking Section
- Dark / Light Theme
- Smooth UI & Responsive Design

# 🚀 Getting Started

## 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/pharaohslegacy.git
```

---

## 2. Navigate into the project

```bash
cd pharaohslegacy
```

---

## 3. Install dependencies

```bash
npm install
```

---

## 4. Run development server

```bash
npm run dev
```
